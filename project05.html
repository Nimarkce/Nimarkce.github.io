Some Commonly Used Bioinformatics Software

Here are some of the bioinformatics software I have used and my experience at using that. The page is still in construction, and I will add more information about them.

Cell2loc

Cell2loc is used to deconvolution the spatial transcriptomics data. It enables us to investigate the cell proportion in a spatially resolved manner

Tangram

Tangram is also a great tool for the spatial deconvolution.

Seurat

A commonly used tool for the single-cell analysis. Basically doing anything. Many tools also based on that. It seems that Seurat is better known than Georges Pierre Seurat

Scanpy

Like Seurat. Also commonly used in single cell analysis. Many tools like Tangram are based on it.

Palantir

For aligning cells in trajectories.

SpaGE

Predict the gene expression level (imputation) in spatial transcriptomics data.

DESeq2

Bioconductor package for differential gene analysis for bulk RNA-seq data.
