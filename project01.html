Cell Segmentation

Cell segmentation is an important step in the pre-processing of the Spatial-Transcriptomics data. Nowadays the technologies of spatial transcriptomics can be mainly classified as two types: Based on NGS and based on image. For these methods that using image to detect the signal of molecular, we need to assign each signal (point at the image) to a given cell. The cell boundary can be marked using H&E staining.

My work focus on how to mark the cell boundary in the image, so that we can assign the signal point in the image into a certain cell. The image then becomes "single cell transcriptomics data with the position". And the data can be used to analysis using the standard workflow of single-cell data.

In detail, I compared three cell-segmentation methods called DeepCell, CellPose and DSB (from Data Science Bowl competition) and selected DeepCell as the best performing tool to do the analysis. Then I make a pipeline for cell-segmentation using DeepCell. Also, noticed that the actual image data by microscope is a image with serval z-stacks (usually in .ome.tiff file format, I tried to apply the segmentation method to the whole z-stacks and to make the same cell in different stack using our method.

Currently I am also trying to use compressed sensing to shorten the time of imaging by using less probes.
